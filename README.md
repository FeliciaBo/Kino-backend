# Kino-hemsida
SSR Movie Website.
Runs on port 5080.

## Stack includes:
- Node.js
- Express
- EJS (for templating)
- Marked (Markdown -> HTML)
- Jest (for testing)
- Supertest (HTTP integration testing)

## How to:

- Install all dependencies:
npm install

- Start server:
npm start

- Start test:
npm test
